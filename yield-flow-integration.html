Características

YieldFlowLoader es un componente React profesional con animaciones GPU-optimizadas, soporte para carga de archivos y seguimiento en tiempo real.

✨ Principales características

  • Animaciones orbitales de cuervos geométricos en 60fps
  • Escudo pulsante con efectos de glow dinámicos
  • Tema oscuro premium con gradientes
  • Barra de progreso en tiempo real (0-100%)
  • Drag & Drop intuitivo
  • Responsive (móvil, tablet, desktop)
  • Totalmente personalizable (colores, velocidades)
  • TypeScript + React 18

Props disponibles

animationSpeed accentColor secondaryColor statusText uploadProgress isLoading onUploadComplete onProgressChange

Uso básico

import { YieldFlowLoader } from './components'; <YieldFlowLoader statusText="UPLOAD" animationSpeed={3} accentColor="#1E90FF" secondaryColor="#24C3A2" />
