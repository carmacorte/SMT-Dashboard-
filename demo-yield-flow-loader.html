💻 Cómo usar

Importa el componente en tu aplicación React:

import { YieldFlowLoader } from './components';

Luego úsalo en tu JSX con las props deseadas. Ver pestaña "Props" para opciones disponibles.

⚙️ Props disponibles

  • animationSpeed (number): 1-10 segundos (default: 3)
  • accentColor (string): Color primario (default: #1E90FF)
  • secondaryColor (string): Color secundario (default: #24C3A2)
  • statusText (string): Texto de estado (default: YIELD FLOW)
  • uploadProgress (number): 0-100% de progreso
  • isLoading (boolean): Mostrar estado de carga
  • onUploadComplete (callback): Al completar carga
  • onProgressChange (callback): Al cambiar progreso
